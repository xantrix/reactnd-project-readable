## Start Developing

To get started developing right away:

* Install and start the API server
    - `cd api-server`
    - `npm install`
    - `node server`
* In another terminal window, install and start the front-end
    - `cd react-ui`
    - `npm install`
    - `npm start`

Or use:

`npm run start-dev`
